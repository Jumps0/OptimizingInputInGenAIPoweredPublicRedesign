
  # Citizen Redesign App Prototype

  Thesis Project for CS-IT9 group to Aalborg University.

  This is a code bundle for Citizen Redesign App Prototype. The original project is available at https://www.figma.com/design/cT5dCMDbY7DMWo0qwYA5zQ/Citizen-Redesign-App-Prototype.

  ## Running the code

  Navigate to the `frontend` folder.

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  (In ssh commands need to be `sudo`)
